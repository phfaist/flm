Overview
========

The **Flexible Latex-Like Markup (FLM)** language is a markup language inspired
by LaTeX syntax.  It provides a simple parser and formatter that renders to
multiple output formats including HTML, plain text, LaTeX, and Markdown.

FLM pushes further one of the core insights of LaTeX, namely, that code should
*describe* document contents in an intuitive way as a markup language, while
disregarding as much as possible the details of how that content will be
typeset.  The final typesetting is fully customizable, e.g., through CSS styling
of its HTML output (including the use of templates).

The framework is designed to be very easily extensible and customizable.  The
parser is based on `pylatexenc <https://github.com/phfaist/pylatexenc>`_ version
3.  FLM syntax is essentially a subset of standard LaTeX commands, including
macros, environments, and some characters that have special meaning; these are
parsed in a loosely similar fashion to usual LaTeX code.

FLM is used to write the contents of the `Error Correction Zoo
<https://errorcorrectionzoo.org/>`_ in a way that is intuitive for scientists,
flexible, and robust.


Why FLM?
--------

- **LaTeX-like, but as markup.**  If you know LaTeX, you already know most of
  FLM.  Write ``\emph{hello}`` for emphasis, ``\section{Title}`` for headings,
  ``\begin{enumerate}...\end{enumerate}`` for lists, and so on.

- **Multi-format output.**  The same FLM source can be rendered as HTML, plain
  text, LaTeX, Markdown, or PDF.

- **Extensible.**  Functionality is organized into *features* --- pluggable
  modules that define macros, environments, and specials.  You can enable,
  disable, configure, or write your own features to tailor the available markup
  to your needs.

- **Configurable.**  Parser settings, renderer options, and feature
  configurations can all be specified via YAML configuration files, YAML front
  matter, or the command line.


Installation
------------

Install the essential FLM Python library with pip::

    pip install flm-core

To get started with the command-line ``flm`` tool along with some convenient
packages to write documents, try::

    pip install flm-core[maincmdl] flm-citations flm-templates flm-htmlplus


Quick Start
-----------

Create a file ``mydocument.flm``:

.. code-block:: latex

    ---
    title: Kitaev's Surface Code
    ---
    \section{Kitaev's Surface Code}

    The \emph{stabilizers} of the \textit{surface code} on the 2-dimensional
    torus are generated by star operators \(A_v\) and plaquette operators
    \(B_p\).  Each star operator is a \textbf{product} of four Pauli-\(X\)
    operators on the edges adjacent to a vertex \(v\) of the lattice; each
    plaquette operator is a product of four Pauli-\(Z\) operators applied to
    the edges adjacent to a face, or plaquette, \(p\) of the lattice
    (\ref{figure:toric-code-operators}).

    \begin{figure}
      \includegraphics{toric-code-operators}
      \caption{Stabilizer generators and logical operators of the 2D surface
        code on a torus.}
      \label{figure:toric-code-operators}
    \end{figure}

Compile it to an HTML page::

    flm mydocument.flm -o mydocument.html --format=html --template=simple

Open ``mydocument.html`` in your browser to see the result.

Available output formats are ``html``, ``text``, ``latex``, and ``markdown``.
You can also generate PDF output with ``--workflow=runlatexpdf --format=pdf`` if
your system has a LaTeX distribution such as `TeX Live <https://tug.org/texlive/>`_
installed.
